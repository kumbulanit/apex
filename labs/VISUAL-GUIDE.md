## üé® Color-Coded Visual Elements

When you see these in the labs, here's what the colors mean in the actual APEX interface:

### Navigation and Buttons

- **Blue Buttons** = Primary actions (Sign In, Create, Save, Apply Changes)
- **Green Buttons** = Success actions (Run, Execute, Deploy)
- **Red Buttons** = Destructive actions (Delete, Remove, Cancel)
- **Gray Buttons** = Secondary actions (Close, Back, Cancel)

### Status Indicators

- **Green Checkmark ‚úÖ** = Success, Active, Enabled
- **Red X ‚ùå** = Error, Inactive, Disabled
- **Orange Warning ‚ö†Ô∏è** = Caution, Review Required
- **Blue Info ‚ÑπÔ∏è** = Information, Help Available

### Component Icons in Page Designer

- **üìù Region** = Content container (Static, Report, Form, Chart)
- **üìä Chart** = Data visualization
- **üìã Report** = Data table
- **üìÑ Form** = Data entry
- **üîò Button** = Action trigger
- **üìù Item** = Input field
- **‚öôÔ∏è Process** = Server-side logic
- **‚ö° Dynamic Action** = Client-side behavior

## üìê Layout Conventions

### Screen Coordinates

When labs reference screen positions:

- **Top-left** = Logo and home navigation
- **Top-center** = Workspace name and breadcrumbs
- **Top-right** = User menu and settings
- **Left sidebar** = Component tree or navigation menu
- **Main content area** = Primary workspace
- **Right sidebar** = Properties or help panel

### Modal Dialogs

Modal dialogs appear centered on screen with:
- **Title bar** at top (often with X close button)
- **Content area** in middle
- **Button bar** at bottom (typically Cancel/OK or Close/Save)

### Standard Navigation Paths

Format: `Starting Point ‚Üí Menu ‚Üí Submenu ‚Üí Destination`

**Common Paths You'll Use:**

1. **Create New Application:**
   ```
   Home ‚Üí App Builder ‚Üí Create ‚Üí New Application
   ```

2. **Create Database Table:**
   ```
   Home ‚Üí SQL Workshop ‚Üí Object Browser ‚Üí Create ‚Üí Table
   ```

3. **Run SQL Query:**
   ```
   Home ‚Üí SQL Workshop ‚Üí SQL Commands ‚Üí [Type Query] ‚Üí Run
   ```

4. **Edit Page:**
   ```
   Home ‚Üí App Builder ‚Üí [Your App] ‚Üí [Page Number] ‚Üí Edit Page
   ```

   ```
   Home ‚Üí App Builder ‚Üí [Your App] ‚Üí Create Page
   ```

   ```
   App Builder ‚Üí [Your App] ‚Üí Run Application (‚ñ∂Ô∏è icon)
   ```

## üí° Pro Tips for Following Visual Instructions

1. **Browser Zoom**: Set browser zoom to 100% for consistent UI
2. **Screen Resolution**: Minimum 1280x720 recommended
3. **Browser Choice**: Chrome, Firefox, or Edge work best
4. **Pop-up Blockers**: Disable for apex.oracle.com
5. **Multiple Tabs**: OK to have multiple APEX tabs open
6. **Auto-Save**: APEX auto-saves most work, but click "Apply Changes" when you see it

## üéØ Quick Reference: Button Actions

| Button Text | What It Does | When to Click |
|-------------|--------------|---------------|
| **Sign In** | Log into workspace | At login screen |
| **Create** | Start creation wizard | When adding new item |
| **Apply Changes** | Save your edits | After changing properties in Page Designer |
| **Run** | Execute SQL or run application | After writing SQL or to test app |
| **Save** | Commit changes | After editing form data |
| **Cancel** | Discard changes | To exit without saving |
| **Delete** | Remove item | To permanently delete |
| **Copy** | Duplicate item | To create similar item |
| **Export** | Download application | To backup or deploy |
| **Import** | Upload application | To restore or install |

## üîç Finding Items on Screen

### If You Can't Find Something:

1. **Check the breadcrumbs** - Shows where you are
2. **Look for search boxes** - Usually top-right of lists
3. **Check filters** - May be hiding items
4. **Scroll down** - Some pages are long
5. **Expand tree nodes** - Click ‚ñ∂ to expand in trees
6. **Check your workspace** - Confirm VODACOM_DEV in top-center

### Common "Can't Find" Issues:

‚ùå **"I don't see App Builder"** ‚Üí You're not on the home page (click logo top-left)

‚ùå **"I don't see my table"** ‚Üí Check you're in Object Browser > Tables

‚ùå **"I don't see my application"** ‚Üí Check you're in App Builder (not SQL Workshop)

‚ùå **"Page Designer looks different"** ‚Üí You may be in old editor (look for 3-panel layout)
