## üéì Study Guide

### Preparation Tips

**For Each Assessment:**
1. Review corresponding lab materials
2. Complete all lab exercises
3. Practice SQL queries in SQL Workshop
4. Recreate sample pages/forms
5. Take practice assessment (if available)

**Time Management:**
- Read all questions first
- Answer easy questions immediately
- Mark difficult questions for review
- Allocate time per section (MC: 1 min/question)
- Leave 5-10 minutes for review

**Common Mistakes:**
- Not reading questions carefully
- Forgetting to use bind variables (SQL injection)
- Confusing authentication vs authorization
- Using correlated subqueries (performance)
- Not testing code before submitting

### Recommended Study Order

**Week 1-2:** Foundations
- Assessment 01: Introduction and Getting Started
- Assessment 02: Creating Applications
- **Practice:** Create 3 applications using different methods

**Week 3-4:** Building Applications
- Assessment 03: Pages and Page Designer
- Assessment 04: Reports and Forms
- **Practice:** Build complete CRUD application

**Week 5-6:** Advanced Features
- Assessment 05: Controls and Navigation
- Assessment 06: Security and Performance
- **Practice:** Add security and optimize existing app

**Week 7:** Deployment
- Assessment 07: Deploying Applications
- **Practice:** Deploy to UAT, create deployment checklist

**Week 8:** Final Preparation
- Review all assessments
- Complete final comprehensive assessment
- **Practice:** Build loyalty program app from scratch

## üîç Answer Key Usage (Instructors)

### Grading Guidelines

**Multiple Choice / True-False:**
- Correct = full points
- Incorrect = 0 points

**SQL Queries:**
- Correct syntax and logic = full points
- Correct logic, minor syntax error = -1 point
- Partially correct = 50% points
- Incorrect or missing = 0 points

**Design Exercises:**
- Complete, functional design = full points
- Missing minor elements = -1 to -2 points
- Incomplete design = 50% points
- Missing or incorrect = 0 points

**Practical Scenarios:**
- All requirements met = full points
- Most requirements met = 70-90% points
- Some requirements met = 50-70% points
- Few requirements met = 0-50% points
